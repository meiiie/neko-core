# Neko Core - Gemini Omni video assets

Attach the numbered files to Gemini in this order:

1. `01-logo-dark.png` - canonical horizontal wordmark on the dark product background.
2. `02-neko-icon.png` - canonical transparent pixel mark. Do not redraw it.
3. `03-avatar-square.png` - square avatar/application icon.
4. `04-terminal-real.png` - real Neko Core v0.11.5 terminal run: todo -> file reads -> verified answer.
5. `05-brand-board.png` - palette, visual language, product architecture and motion reference.

The source HTML for the board is retained as `05-brand-board.html`. The complete storyboard,
voice-over and ready-to-paste Gemini prompt are in
`../../docs/marketing/GEMINI_OMNI_VIDEO_KIT.md`.

Hard rule: use the supplied logo pixels as-is. Do not replace the mark with a generic cat, robot,
sparkle, brain, or purple AI orb.
